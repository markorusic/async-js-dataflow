# JS Async Dataflow

## Project setup

Install node modules:
```sh
npm i
```

Start local server:
```sh
npm run server
```

Run examples:
```sh
node examples/01-fetch.js
```

Note: React code for ```examples/07-react.js``` is located in ```public/index.js```
